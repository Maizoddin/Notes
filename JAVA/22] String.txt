* String class exists in java.lang package
* String is immutable.

* strings are stored in stringpool.
* If we store same string in 2 variables then it will create only one string in string pool and both the variables will point to that.
Example :
String role = "admin";
String position = "admin";

If we use new keyword then it will create 2 different string objects.
Example :
string name = new String("John");
