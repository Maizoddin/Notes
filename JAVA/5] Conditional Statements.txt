Conditional :
? :

Note : It must return a result of the expression.

Valid :
String message = score > 90 ? "Good" : "Bad"
OR
System.out.println(score > 90 ? "Good" : "Bad") 

Invalid :
score > 90 ? System.out.println("Good") : System.out.println("Bad");
