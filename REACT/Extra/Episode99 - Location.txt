Fetching the location of the user
navigator.geolocation.getCurrentPosition() is a method provided by the Geolocation API in web browsers. It's used to retrieve the current geographic location of the user.

Example :
navigator.geolocation.getCurrentPosition((position) => {
  const latitude = position.coords.latitude;
  const longitude = position.coords.longitude;
  console.log(`Latitude: ${latitude}, Longitude: ${longitude}`);
}, (error) => {
  console.error(`Error: ${error.message}`);
});


Now once we get the latitude and longitude and we can convert this to human readable location using process called "reverse geocoding"
Reverse geocoding is the process of converting geographic coordinates into a human-readable address.
so here we can use GoogleMaps API where we can send latitude and longitude and can get the exact location.
Example :
`https://maps.googleapis.com/maps/api/geocode/json?latlng=${latitude},${longitude}&key=YOUR_API_KEY`

Code :
navigator.geolocation.getCurrentPosition((position) => {
  const latitude = position.coords.latitude;
  const longitude = position.coords.longitude;

  fetch(`https://maps.googleapis.com/maps/api/geocode/json?latlng=${latitude},${longitude}&key=YOUR_API_KEY`)
    .then((response) => response.json())
    .then((data) => {
      if (data.status === "OK") {
        const address = data.results[0].formatted_address;
        console.log("Current Location:", address);
      } else {
        console.error("Failed to fetch address.");
      }
    })
    .catch((error) => {
      console.error("Error fetching address:", error);
    });
}, (error) => {
  console.error(`Error: ${error.message}`);
});
