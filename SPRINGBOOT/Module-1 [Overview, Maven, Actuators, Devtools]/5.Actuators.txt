Spring Boot Actuator :
Spring Boot Actuator provides endpoints to monitor and manage applications.

Common Actuator Endpoints :
/actuator: Lists all available endpoints.
/actuator/health: Shows application health.
/actuator/info: Displays custom application info.
/actuator/metrics: Provides application metrics.
/actuator/mappings: Lists all Spring MVC endpoints.

Steps to Enable Actuator :
1] Add the dependency:
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-actuator</artifactId>
</dependency>

2] Configure in application.properties :
management.endpoints.web.exposure.include = health,info,mappings
management.info.env.enabled = true

Note : 
To expose all endpoints :
management.endpoints.web.exposure.include = *

---------------------------------------------

Customizing Actuator Info :
management.info.app.enabled=true
management.info.java.enabled=true
management.info.env.enabled=false

info.app.name=BankingApp
info.app.version=2.0

Example Output (JSON):
{
  "app": {
    "name": "BankingApp",
    "version": "2.0"
  },
  "java": {
    "version": "21"
  }
}

---------------------------------------------

Disabling Specific Actuator Endpoints :
management.endpoints.web.exposure.exclude=info,health,mappings

---------------------------------------------

Adding Security to Actuator Endpoints :
1] Add dependency:
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
</dependency>

2] Configure credentials in application.properties:
spring.security.user.name=admin
spring.security.user.password=secret

---------------------------------------------
